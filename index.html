Jigsaw Puzzle Solving Using Robotic Arm

Our approach was to first take a picture of an unsolved jigsaw puzzle and solve a digital representation of the puzzle in software before translating the necessary software generated movements into physical movements that would be implemented by a robotic arm. Through our research, we were able to create a robust algorithm for solving the jigsaw puzzle with a digitally generated representation.

House Hack

With this project we aimed to decentralize the housing market using blockchain technology. We store contracts to houses on the chain, which users can buy and sell houses that have been verified to appear on the chain.
